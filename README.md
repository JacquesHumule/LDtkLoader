# LDtkLoader

### Load LDtk levels in your C++ game

**LDtkLoader** is a loader for [LDtk](https://github.com/deepnight/ldtk) tile maps.

WIP, API is not definitive.

```c++
ldtk::World world;
world.loadFromFile("my_levels.ldtk");
```
